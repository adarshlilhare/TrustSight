Triple-Layer Protection

Three independent security engines working together to keep your browser safe.

🛡️

Pre-Navigation Blocking

Intercepts malicious URLs before the page even loads. Checks against 150+ known threat domains and suspicious URL patterns.

🧠

Edge AI Phishing Detection

On-device AI engine scores page content for phishing keywords, suspicious forms, hidden iframes, and brand impersonation attempts.

🚫

Ad & Tracker Blocking

DOM-level blocking using 30+ CSS selectors and MutationObserver. Strips tracker scripts from Google Analytics, Facebook Pixel, and more.

📋

Allow List Engine

Whitelist trusted domains via the Dashboard or block page bypass. Allowed domains instantly skip threat scanning with zero latency.

🔒

Secure Bypass

URL validation via new URL() constructor and location.replace() prevents Open Redirect vulnerabilities when bypassing blocks.

📡

100% Offline

All threat databases, AI scoring, and settings stored locally. Your browsing data never leaves your device — zero telemetry.

Install in 4 Steps

1

Clone the Repository

git clone https://github.com/adarshlilhare/TrustSight.git

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your browser

3

Enable Developer Mode

Toggle "Developer mode" switch in the top right corner

4

Load Unpacked

Click "Load unpacked" and select the TrustSight folder. Done!
